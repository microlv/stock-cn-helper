# stock-cn-helper

中国股票分析助手（前端静态版，可部署 GitHub Pages）。

## 功能

- 实时拉取A股行情（腾讯行情接口）
- 行业分类 + 动量/估值/活跃度/稳定性综合评分
- 三种风险偏好权重（平衡 / 成长 / 价值）
- 推荐列表（A/B/C 分层）
- 全量表格查看与筛选
- 指标学习卡（新手友好）
- 默认每60秒自动刷新一次

## 数据说明

当前为实时行情驱动版本，核心行情字段来自公开行情接口。若接口临时不可用，页面会提示重试。
