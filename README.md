# stock-cn-helper

中国股票分析助手（前端静态版，可部署 GitHub Pages）。

## 功能

- 行业归一化指标评分（PE / PB / ROE / 增速 / 负债率 / 股息率）
- 三种风险偏好权重（平衡 / 成长 / 价值）
- 推荐列表（A/B/C 分层）
- 全量表格查看与筛选
- 指标学习卡（新手友好）
- 支持导入 CSV 数据进行自定义分析

## CSV格式

```csv
code,name,industry,pe,pb,roe,revGrowth,debt,dividend
600519,贵州茅台,白酒,28,9.8,33,15,21,1.8
```
